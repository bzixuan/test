# OpenClaw 中文汉化包

这是 OpenClaw 工作空间的中文汉化版本。

## 文件对照表

| 英文文件 | 中文文件 | 说明 |
|---------|---------|------|
| AGENTS.md | AGENTS_zh.md | 工作空间指南和代理行为规范 |
| SOUL.md | SOUL_zh.md | 核心身份和价值观 |
| USER.md | USER_zh.md | 用户信息模板 |
| IDENTITY.md | IDENTITY_zh.md | 身份定义模板 |
| BOOTSTRAP.md | BOOTSTRAP_zh.md | 首次启动指南 |
| TOOLS.md | TOOLS_zh.md | 本地工具笔记 |
| HEARTBEAT.md | HEARTBEAT_zh.md | 心跳任务配置 |

## 使用方法

### 方法一：替换文件
将中文文件（`*_zh.md`）的内容复制到对应的英文文件中。

```bash
cp AGENTS_zh.md AGENTS.md
cp SOUL_zh.md SOUL.md
cp USER_zh.md USER.md
cp IDENTITY_zh.md IDENTITY.md
cp BOOTSTRAP_zh.md BOOTSTRAP.md
cp TOOLS_zh.md TOOLS.md
cp HEARTBEAT_zh.md HEARTBEAT.md
```

### 方法二：保留双语
保留原有英文文件，同时保留中文文件作为参考。
